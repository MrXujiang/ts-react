该项目基于 [Create React App](https://github.com/facebook/create-react-app)构建.

## 运行

在项目目录中, 你能执行以下脚本运行项目:

### `npm start`

在开发模式下运行.<br>
在浏览器里打开 [http://localhost:3000](http://localhost:3000).

页面会根据你的修改而实时更新.<br>
你还可以在控制台看到脚本中的错误.

### `npm test`

应用测试

### `npm run build`

打包项目以便提交到生产环境.<br>
